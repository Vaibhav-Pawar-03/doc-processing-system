# Document Processing System

## 🚀 Tech Stack
- Frontend: Next.js
- Backend: FastAPI
- Queue: Celery + Redis
- DB: PostgreSQL
- Dockerized

## ⚡ Features
- File upload
- Async processing
- Real-time status via WebSocket
- Download processed files

## 🛠 Run Locally

### Backend
docker-compose up --build

### Frontend
cd frontend
npm install
npm run dev
